自！己！找！(暴躁)

sudo aircrack-ng -w Desktop/wordlist.txt Desktop/handshake.cap



// ircrack-ng -w <字典路径> <握手包路径>

破解的速度取决于信号强的，CPU运算速度等等，但是最重要的，还是要看老王家的WIFI密码的强度，像123456这种基本上几秒钟就出来了

那么这个时候很多同学会问了，字典路径是啥？ 首先，各位需要了解的是，这个破解方法是暴力破解，事先设置好字典然后一个一个试。比如 字典里面的数值为 1 2 3 11 22 33

那么破解就根据顺序，先用密码 1 连接wifi，密码错误，那么就用2 3 11 22……很无脑，但是是最简单的破解方法

==字典自己找==

或者

安装crunch，输入

apt-get install crunch

安装完成后

crunch <min> max<max> <characterset> -t <pattern> -o <output filename>

==min= 最小密码长度。  max= 最大密码长度。  characterset= 用于生成密码的字符集。  pattern= 生成的密码的指定模式。  outputfile= 保存字典文件的路径。去掉<>==

注： pattern中可以使用的特殊标记字符(ex.3) 

[%]插入数字 

[@] 插入小写字母 

[,] 插入大写字母 

[^] 插入符号

如果你知道目标的生日是0728（7月28日），并且认为他们以自己的密码包含他们生日，则可以生成一个以0728为结尾的密码列表，方法为@@@@@@@0728。 该字最多可生成11个字符（7个变量和4个固定）密码，然后全部都以0728结尾

crunch 8 8 0123456789 -o Desktop/wordlist.txt
=

字典中只包含长度为8的0-9字符集

生成完字典后，就可以拿来用了 输入最开头的代码

sudo aircrack-ng -w Desktop/wordlist.txt Desktop/handshake.cap

之后的窗口会显示破解字典的进度，这里我就不上图了 如果破解成功，会提示

注意事项

1监听窗口和破解窗口都必须一直开着且运行

2破解开始前要确保在监听的情况下，捕捉到连接动作。就是说，有人连接WiFi且你在监听

3监听必须开抓包，并且破解的那行代码的最后一小段指向抓包抓下来的cap文件，文件名可能有所不同，请自己衡量

4wpa/wp2破解成功率不高，实际情况看缘分
